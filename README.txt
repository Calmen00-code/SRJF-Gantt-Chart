This is a program that illustrates the gantt chart for a CPU Scheduling. 
The CPU scheduling implemented was Shortest Job Remaining Scheduling.

Compiling Program,
    make

Running Program,
    ./main

    After running ./main, the program will print "INPUT" in the console screen. 
    Which will enable the user to input the file name which contains the processes.

Inputting File Name,
    INPUT: tasks.txt

Quiting Program,
    INPUT: QUIT
